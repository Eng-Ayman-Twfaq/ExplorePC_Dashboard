## 🌟 نظام إدارة ExplorePC

لوحة تحكم متكاملة لإدارة منصتي:
- 🛍️ **تطبيق العملاء** (واجهة شراء المنتجات)
- 🏪 **تطبيق التاجر** (إدارة المخزون والطلبات)
- ☁️ **الخدمات السحابية** (Firebase, MySQL)
